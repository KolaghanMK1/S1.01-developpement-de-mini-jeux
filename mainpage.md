# Dev_menu_jeux Documentation

This documentation was generated using Doxygen. It documents the C/C++ sources in the `Programmes` workspace without modifying any source files.

How to generate the documentation (Windows):

1. Install Doxygen: https://www.doxygen.nl/download.html
2. Open a PowerShell in the `Programmes` folder (where `Doxyfile` is located)
3. Run the `run_doxygen.bat` script included in this repository, or run `doxygen Doxyfile` manually.

Output will be placed in `docs/doxygen/html`.

Notes:
- The Doxygen config is set to `EXTRACT_ALL` so documentation will be generated even if there are no Doxygen comments in the sources.
- The generated HTML contains a tree view and source browsing where supported.
